## 大概流程

实现的是带优先级的时间片轮转调度。就是，根据进程优先级轮着来时间片...

主要是三个部分 OS 类，CPU 类，GUI 类

GUI 负责绘制界面，CPU 负责处理计算逻辑，OS 负责调度及中断处理等操作。

main.py 是主程序，

```python 
python main.py
```
